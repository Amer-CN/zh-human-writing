# references/protected-spans.md
# zh-human-writing v1 — Protected Spans 定义

---

## 1. 事实性内容

### 数字
- 检查方式：逐字比对
- 允许：千分位格式变化（1,000 ↔ 1000）
- 不允许：数值变化（100 → 99）

### 日期
- 检查方式：逐字比对
- 允许：YYYY-MM-DD ↔ YYYY年M月D日；时区 +08:00 ↔ 北京时间
- 不允许：日期值变化

### 百分比
- 检查方式：逐字比对
- 允许：50% ↔ 50％
- 不允许：数值变化

### URL
- 检查方式：逐字比对
- 允许：http ↔ https（如原文已有）；末尾斜杠有无
- 不允许：URL 路径变化、域名变化

### 人名、机构名、产品名
- 检查方式：逐字比对
- 不允许：任何变化

---

## 2. 技术性内容

| 类别 | 检查方式 | 允许 | 不允许 |
|------|---------|------|--------|
| 代码块 | 逐字比对 | 无 | 代码内容任何变化 |
| 命令 | 逐字比对 | 无 | 命令任何变化 |
| 路径 | 逐字比对 | / ↔ \ | 路径内容变化 |
| 参数 | 逐字比对 | 无 | 任何变化 |
| 字段名 | 逐字比对 | 无 | 任何变化 |
| 日志 | 逐字比对 | 无 | 任何变化 |
| 错误码 | 逐字比对 | 无 | 任何变化 |
| 状态码 | 逐字比对 | 无 | 任何变化 |

---

## 3. 语义性内容

### 引用原文
- 检查方式：逐字比对
- 不允许：引用原文的任何变化

### 否定
- 检查方式：语义比对
- 不允许：否定消失或反转

### 条件
- 检查方式：语义比对
- 不允许：条件消失或改变

### 因果
- 检查方式：语义比对
- 不允许：因果关系消失或反转

### 责任主体
- 检查方式：语义比对
- 不允许：观点归属转移

### 不确定程度
- 检查方式：语义比对
- 不允许："可能"变成"确定"

---

## 4. 作者性内容

- 作者明确指定不可改的片段：逐字比对
- 作者的个人观点和立场：语义比对
- 作者的语气和语域：语体比对

---

## 5. 用户显式 protected spans

### 标记方式

用户可以使用以下方式标记不可改的片段：

- 使用 `[[protected]]...[[/protected]]` 标记
- 使用 `<!--keep-->...<!--/keep-->` 标记
- 在输入参数中列出位置范围

### 检查方式

逐字比对。用户标记为不可改的片段不得有任何变化。

---

## 检查工具

确定性检查由 `scripts/fidelity_guard.py` 执行。
语义性检查由 LLM 语义判断执行，脚本只负责格式化输出。
