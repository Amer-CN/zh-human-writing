## 2. 缺陷详情

### 缺陷 AHR-004: 聊天助手残留模式漏检

| 项目 | 说明 |
|------|------|
| **发现阶段** | 对抗性测试 — hard-residue 类 |
| **发现测试** | AHR-004: "希望这对你有帮助！如果还有其他问题，请随时告诉我。" |
| **严重程度** | 高（hard-residue 级别，影响 pass/fail 判定） |
| **根因** | `pattern_audit.py` 中 HR-004 的正则模式列表不完整，缺少以下常见 AI 助手结尾语：|
| | - `希望这对你有帮助` |
| | - `如果还有其他问题` |
| | - `请随时告诉我` |
| **影响** | 包含上述短语的文本不会触发 hard-residue 检测，导致 pass/fail 误判为 pass |
| **波及测试** | AHR-004（直接失败）、ALF-001（长文保护测试因 hr_count=0 间接失败） |

#### 修复前代码

```python
{
    'id': 'HR-004',
    'name': '聊天助手残留',
    'patterns': [r'请问还有什么可以帮助', r'还有什么我可以帮助', r'如果您有其他问题', r'还有什么我可以为您'],
    'language_origin': 'language_general',
},
```

#### 修复后代码

```python
{
    'id': 'HR-004',
    'name': '聊天助手残留',
    'patterns': [r'请问还有什么可以帮助', r'还有什么我可以帮助', r'如果您有其他问题', r'还有什么我可以为您',
                 r'希望这对你有帮助', r'如果还有其他问题', r'请随时告诉我'],
    'language_origin': 'language_general',
},
```

#### 同步更新的文档

`references/patterns/hard-residue.md` 中 HR-004 的触发线索已同步更新：

```
- **触发线索**: "请问还有什么可以帮助您的？"、"还有什么我可以帮助的吗？"、"如果您有其他问题"、
  "希望这对你有帮助"、"如果还有其他问题"、"请随时告诉我"
```
